# RAK's Microsoft Activation Scripts (MAS) v3.9

## About This Project

**Microsoft Activation Scripts (MAS)** is a powerful all-in-one activation solution built and maintained by **RAK's Corporation**.

This script provides multiple activation methods for:
- **Windows 10/11** (HWID & TSforge)
- **Microsoft Office** (Ohook & TSforge)
- **Windows ESU** (TSforge)

---

## Features

### Activation Methods

1. **HWID** - Hardware ID activation for Windows 10/11
2. **Ohook** - Office License Service activation
3. **TSforge** - Advanced activation for Windows, Office & ESU
4. **Online KMS** - Key Management Service activation

### Additional Features

- Check Activation Status
- Change Windows Edition
- Change Office Edition
- Troubleshooting Tools
- OEM Folder Extraction
- Download Genuine Windows/Office ISO

---

## How to Use

### Basic Usage

1. Right-click `MAS_AIO.cmd`
2. Select **"Run as administrator"**
3. Choose your activation method from the menu

### Command Line Options

```batch
MAS_AIO.cmd /HWID          - Activate with HWID (Windows)
MAS_AIO.cmd /Ohook         - Activate with Ohook (Office)
MAS_AIO.cmd /Z-Windows     - Activate with TSforge (Windows)
MAS_AIO.cmd /K-WindowsOffice - Activate with Online KMS
```
